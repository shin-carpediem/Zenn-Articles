# Zenn-Articles

zenn に投稿する記事を管理します。

### Zenn CLI

#### [📘 使い方](https://zenn.dev/zenn/articles/zenn-cli-guide)

- 新しい記事を作成する

```
$ npx zenn new:article
```

- 新しい本を作成する

```
$ npx zenn new:book
```

- 投稿をプレビューする

```
$ npx zenn preview
```

### マークダウン記法/特筆

- 画像を埋め込む

```
![](/images/${記事のユニークID(例：f6ac0f5f0b72e3)}/${画像ファイル名(例：hoge.png)})
```
